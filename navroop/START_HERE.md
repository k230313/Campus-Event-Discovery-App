## 🎮 How to Test:

### 1. Not Logged In:
- Browse events
- See About/Contact pages
- Try login/register

### 2. As Student:
- Register with role: **Student**
- Browse and RSVP to events
- Bookmark events
- View My Events & Bookmarks
- Unregister from events

### 3. As Organizer:
- Register with role: **Organizer**
- Create new events
- Manage your events
- Edit/delete events

### 4. As Admin:
- Register with role: **Admin**
- View admin dashboard
- Edit users
- Manage all events
- View reports

## 🤖 AI Chatbot:

2. Copy `.env.example` to `.env`
3. Add: `VITE_OPENAI_API_KEY=sk-your-key`
4. Restart server
